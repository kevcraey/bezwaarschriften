# TDR-007: Testcontainers voor integratietests

**Datum:** 2026-01

## Context

Integratietests moeten tegen een echte PostgreSQL + pgvector database draaien. Een gedeelde testdatabase is fragiel en niet reproduceerbaar.

## Besluit

Testcontainers start een disposable PostgreSQL container met pgvector extensie per testrun. Docker daemon vereist.

## Consequentie

- Elke testrun begint met een schone database.
- Geen externe testinfrastructuur nodig.
- Docker daemon moet draaien; tests zijn trager dan pure unit tests.
