# ADR-005: Lit + @domg-wc design system voor frontend

**Datum:** 2026-01

## Context

De applicatie is een interne tool voor de Vlaamse overheid. Er bestaat een verplicht design system (Vlaanderen Flux / @domg-wc) gebaseerd op Lit web components.

## Besluit

Alle UI-componenten bouwen met Lit en uitsluitend @domg-wc componenten gebruiken. Geen standaard HTML-elementen als er een @domg-wc equivalent bestaat. Custom CSS is uitzonderlijk.

## Consequentie

- Consistente look-and-feel conform Vlaamse huisstijl.
- Beperkte flexibiliteit: enkel beschikbare componenten uit de bibliotheek.
- Shadow DOM vereist speciale aanpak voor event handling en styling.
