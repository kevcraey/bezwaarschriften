# TDR-001: Java 21 + Spring Boot 3.4

**Datum:** 2025-12

## Context

Keuze van runtime en framework voor een nieuwe applicatie die AI-integratie, database-toegang en web-API's combineert.

## Besluit

Java 21 (LTS) met Spring Boot 3.4. Java 21 biedt virtual threads en records. Spring Boot 3.4 is de eerste versie met stabiele Spring AI ondersteuning.

## Consequentie

- Virtual threads mogelijk voor IO-intensieve AI-calls.
- Records voor immutable DTOs (minder boilerplate).
- Spring AI 1.0.x beschikbaar als first-class dependency.
