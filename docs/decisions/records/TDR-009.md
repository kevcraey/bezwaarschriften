# TDR-009: JPA voor data-access

**Datum:** 2025-12

## Context

De applicatie heeft zowel complexe relationele queries als eenvoudige CRUD-operaties nodig.

## Besluit

Spring Data JPA met Hibernate als ORM. Standaard repositories voor CRUD, native queries enkel voor vector-operaties (`<=>` operator) die JPA niet ondersteunt.

## Consequentie

- Snelle ontwikkeling van standaard data-access.
- Native queries nodig voor pgvector-specifieke operaties.
- Hibernate entity lifecycle moet begrepen worden (lazy loading, flush timing).
