# TDR-011: Cypress voor frontend tests

**Datum:** 2026-02

## Context

Frontend gebruikt Lit web components met shadow DOM. Standaard testing libraries (Jest + jsdom) ondersteunen shadow DOM slecht.

## Besluit

Cypress component tests voor Lit web components. Cypress rendert componenten in een echte browser en kan shadow DOM traverseren.

## Consequentie

- Betrouwbare tests die shadow DOM correct afhandelen.
- Cypress vereist een browser-runtime (trager dan jsdom).
- Component tests draaien headless in CI.
