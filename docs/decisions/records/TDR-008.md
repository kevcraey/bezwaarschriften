# TDR-008: UMAP voor dimensiereductie

**Datum:** 2026-02

## Context

HDBSCAN werkt slecht op high-dimensional data (1024-dim embeddings). Dimensiereductie is nodig als voorstap.

## Besluit

UMAP (Uniform Manifold Approximation and Projection) reduceert embeddings van 1024 naar 2 dimensies. Behoudt lokale structuur beter dan t-SNE en is sneller.

## Consequentie

- HDBSCAN krijgt low-dimensional input en clustert effectief.
- UMAP-parameters (n_neighbors, min_dist) beïnvloeden clusterresultaten.
- Java-implementatie via UMAP-jvm bibliotheek.
