# TDR-006: Liquibase voor database migraties

**Datum:** 2025-12

## Context

Database-schema moet reproduceerbaar evolueren over omgevingen (dev, test, productie).

## Besluit

Liquibase changelogs in XML-formaat. Elke schemawijziging als apart changeset met uniek ID. Automatisch uitgevoerd bij applicatie-start.

## Consequentie

- Schema-evolutie is versiebeheerd en reproduceerbaar.
- Rollback mogelijk via Liquibase mechanismes.
- Changelogs moeten zorgvuldig geordend worden om conflicten te vermijden.
