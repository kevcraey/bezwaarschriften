# TDR-004: bge-m3 embedding model

**Datum:** 2026-01

## Context

Bezwaarteksten zijn Nederlandstalig. Veel embedding modellen zijn primair op Engels getraind.

## Besluit

bge-m3 als embedding model. Ondersteunt 100+ talen waaronder Nederlands, produceert 1024-dimensionale embeddings met goede meertalige semantische kwaliteit.

## Consequentie

- Goede resultaten op Nederlandse tekst zonder fine-tuning.
- Groot model (meer resources dan bv. MiniLM).
- Lokaal te draaien via Ollama voor ontwikkeling.
